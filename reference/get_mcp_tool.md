# Get an MCP tool from a list of tool definitions (convenience function)

Get an MCP tool from a list of tool definitions (convenience function)

## Usage

``` r
get_mcp_tool(tool_defs, tool_name)
```

## Arguments

- tool_defs:

  List. A list of tool definitions

- tool_name:

  Character. The name of the tool to get

## Value

The tool definition
